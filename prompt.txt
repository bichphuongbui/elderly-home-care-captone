Tạo file `PricingPage.tsx` để hiển thị bảng giá ước lượng dành cho Guest (người chưa đăng nhập).

---

🧩 Giao diện gồm:
1. Phần giới thiệu:
   - Tiêu đề: “Ước tính chi phí chăm sóc người cao tuổi”
   - Mô tả ngắn: “Chọn thời lượng chăm sóc để xem giá ước lượng. Việc thanh toán sẽ do bạn và người chăm sóc tự thỏa thuận.”

2. Form chọn thông tin:
   - Input chọn số giờ chăm sóc mỗi ngày (1–24 giờ)
   - Input chọn số ngày chăm sóc (1–30 ngày)
   - Button “Tính chi phí”

3. Phần kết quả:
   - Hiển thị giá ước lượng theo công thức:
     ```
     estimatedPrice = hoursPerDay * days * unitPrice
     ```
   - `unitPrice` có thể là `100,000 VNĐ / giờ`

4. Có thể thêm 3 gói minh họa:
   - Gói Cơ bản: 2h/ngày
   - Gói Tiêu chuẩn: 4h/ngày
   - Gói Toàn thời gian: 8h/ngày

---

🧠 Logic:
- Dùng `useState` để lưu giá trị input và kết quả
- Tự động tính lại khi input thay đổi (hoặc nhấn nút)
- Hiển thị kết quả rõ ràng: “Ước tính chi phí: 6,000,000đ”

---

💄 UI yêu cầu:
- Canh giữa nội dung
- Font lớn, dễ đọc
- Dùng Tailwind với card, input, button rõ ràng
- Responsive trên mobile
