1. Tạo file `blog.service.ts` trong `src/service/`

Export 2 hàm:
- `getAllBlogs(): Promise<Blog[]>` → gọi GET https://68aed258b91dfcdd62ba657c.mockapi.io/blogs
- `getBlogById(id: string): Promise<Blog>` → gọi GET https://68aed258b91dfcdd62ba657c.mockapi.io/blogs/:id

Định nghĩa type `Blog` gồm: id, title, content, createdAt, thumbnail

2. Tạo trang `BlogListPage.tsx`:

- Gọi `getAllBlogs()` từ service
- Hiển thị danh sách blog theo card layout:
  + Ảnh thumbnail, tiêu đề, ngày đăng
  + Nút "Xem chi tiết" → điều hướng đến `/blog/:id`
- Dùng TailwindCSS, layout responsive, đẹp

3. Tạo trang `BlogDetailPage.tsx`:

- Gọi `getBlogById(id)` từ service (lấy id từ URL param)
- Hiển thị:
  + Ảnh thumbnail lớn
  + Tiêu đề
  + Thông tin ngày đăng
  + Nội dung chi tiết (content)
- TailwindCSS đẹp, dễ đọc

4. Cập nhật `routes/index.tsx`:

- Thêm:
```tsx
<Route path="/blog" element={<BlogListPage />} />
<Route path="/blog/:id" element={<BlogDetailPage />} />
