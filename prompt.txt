Tạo file `ChatPage.tsx` trong `pages/caregiver/` để hiển thị giao diện chat giữa Caregiver và Care Seeker.

Yêu cầu:
1. Layout chia 2 phần:
   - Sidebar bên trái (w-1/4): danh sách Care Seeker đang trò chuyện
   - Chat box bên phải (w-3/4): hiển thị hội thoại với người được chọn

2. Sidebar:
   - Hiển thị tên Care Seeker
   - Click vào Care Seeker → load messages của họ vào chat box
   - Highlight Care Seeker đang chọn

3. Chat box:
   - Header: tên Care Seeker
   - Body: danh sách tin nhắn
     + Tin nhắn Care Seeker align left, bg-gray-200
     + Tin nhắn Caregiver align right, bg-blue-500 text-white
   - Footer: input + nút gửi
     + Nhập tin nhắn → thêm vào state messages
     + Clear input sau khi gửi

4. Mock data:
   - Dùng mảng conversations như đã mô tả
   - Lưu state cho selectedConversation
   - Khi gửi tin mới → push vào messages của conversation hiện tại

5. UI:
   - Dùng TailwindCSS
   - Card bo góc, shadow nhẹ
   - Responsive: trên mobile sidebar ẩn, chỉ hiển thị chat
