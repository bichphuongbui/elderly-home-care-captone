Tạo component `AdminLayout.tsx` bằng React + TypeScript.

Yêu cầu:
- Hiển thị layout dạng 2 phần:
  1. Sidebar dọc bên trái, chiều cao full screen
  2. Khu vực nội dung chính bên phải (`children`)

Sidebar:
- Có thể đóng/mở (state toggle)
- Khi mở: hiển thị icon + label menu
- Khi đóng: chỉ còn icon
- Menu gồm:
  + Dashboard
  + Quản lý người dùng
  + Duyệt hồ sơ caregiver
  + Phản hồi / Đánh giá
  + Đăng xuất
- Dùng icon từ `react-icons` (nếu có thể)
- Styling bằng TailwindCSS:
  + Sidebar: bg-gray-800, text-white, rounded, có transition
  + Nút toggle: đặt ở trên cùng sidebar
  + Responsive tốt

AdminLayout nhận `children` làm prop, hiển thị nội dung trang trong khu vực bên phải

Export từ file `layouts/AdminLayout.tsx`
