Chỉnh sửa hoặc tạo mới file `UserManagementPage.tsx` bằng React + TypeScript, dùng TailwindCSS để hiển thị danh sách người dùng từ API thật.

Yêu cầu:
1. Gọi dữ liệu từ API:
  - Dùng Axios để fetch dữ liệu từ:
    https://68aed258b91dfcdd62ba657c.mockapi.io/users
  - Gọi trong `useEffect`, lưu vào state `users`

2. Hiển thị dữ liệu dạng bảng:
  - Cột: Họ tên, Email, Vai trò (role), Trạng thái (mock: active/blocked), Hành động
  - Dòng có hover effect, bo góc nhẹ
  - Responsive tốt, có scroll ngang nếu bảng quá rộng

3. Hành động:
  - Nút "Xem": mở modal hoặc alert hiển thị thông tin chi tiết user
  - Nút "Khoá" / "Mở khoá": toggle trạng thái trong local state (chưa cần gọi API PUT)

4. Giao diện:
  - Dùng TailwindCSS cho bố cục hiện đại, rõ ràng
  - Nút "Xem" màu indigo, "Khoá" màu đỏ nhạt, "Mở khoá" màu xanh
  - Có loading indicator khi đang fetch data
  - Tiêu đề trang: "Quản lý người dùng"

5. Dữ liệu `status` (active/blocked) có thể gán ngẫu nhiên (tạm thời) vì API chưa có field này

6. Export từ file `pages/UserManagementPage.tsx`
